- [x] **하이브리드 랭킹**: BM25 + TF-IDF 가중 결합

  - 요구사항에 없지만 검색 성능 향상에 도움
  - 과도하지 않음 ✅

- [x] **UI 개선**: 페이지네이션, 검색 히스토리, 쿼리 분석
  - 사용자 경험 향상
  - 과도하지 않음 ✅

### 과도하지 않은 이유
1. **하이브리드 랭킹**: 기존 BM25와 TF-IDF를 단순 결합한 것으로, 새로운 알고리즘을 발명한 것이 아님
2. **쿼리 확장**: 간단한 사전 기반 및 통계 기반 확장으로, 복잡한 신경망 모델을 사용하지 않음
3. **Cross-Encoder**: 특강 자료에서 권장하는 용도(Reranking)로만 사용
